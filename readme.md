### Question: 1.c
**Input**
- 2 Lines
- First Line: N
- Second Line: N - 1
- 2 <= N <= 1,000,000

**Result**
Find the missing part

**Example: 1**    
**In**
```
4
2 1 3
```

**Out**
```
4
```

**Example: 2**    
**In**
```
6
6 3 4 1 2
```

**Out**
```
5
```

### Question: 4.c
**Input**
- Hold the Input
- Two Blocks of Code
    - First Block: While / If
    - Second Block: Do_While / Switch_Case
- Process Inputs:
    - 0 - 10  => 10
    - 11 - 20 => 25
    - More than 20 => 50
- First -1  = Switch
- Second -1 = End

**Example**    
**In**
```
5
18
98
-1
21
10
11
20
-1
```
**Out**
```
code_1:
10
25
50
code_2:
50
10
25
25
```
